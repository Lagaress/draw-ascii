## About

A library to draw ASCII art animals, tools, and food

## Installation

```bash
npm install  draw-ascii
```

## Usage

```javascript
const { cow } = require('draw-ascii');

console.log(cow);

// Output
//       \\   ^__^
//        \\  (oo)\\_______
//           (__)\\       )\\/\\
//               ||----w |
//               ||     ||
```
